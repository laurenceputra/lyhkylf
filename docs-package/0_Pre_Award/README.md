## How to use
1) Applicants complete PA-0 first (hard gates).
2) Submit PA-1 (EOI) for initial screen.
3) If invited, complete PA-2 through PA-6.
4) Attach PA-4 and PA-5 as annexes to PA-2, and attach PA-3 as the budget appendix.

## Notes
- The Trajectory Lever is an anchored open set: choose a default lever or define a custom lever that passes the Anchor Test.
- Measurement should remain low-burden; ≤3 instruments is preferred.
